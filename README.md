# IIB Project: Analysis of Low-Cost Particle Image Velocimetry

This repository contains the code and supporting material for my IIB project on the analysis of low-cost **Particle Image Velocimetry (PIV)** systems.

## Repository Contents

### `Analysis of Low Cost Particle Velocimetry.pdf`

A Master's thesis describing the development, methodology, and evaluation of a low-cost Particle Image Velocimetry system. The thesis also explains the architecture and implementation of the accompanying software.

### `DigitalTwin.py`

A Python module for:

* Synthetic PIV image generation
* Velocity field estimation
* System error analysis

The design and underlying methodology are described in the accompanying thesis.

### `DataGeneration.py`

A script used to generate the majority of the datasets presented in the thesis. It also serves as an example of how to use the `DigitalTwin.py` module.

### `Data Viewer.py`

A collection of data visualisation tools designed for exploring and analysing the datasets generated by `DataGeneration.py`.
